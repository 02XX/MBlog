# 嵌套类

| 声明位置 | 包含它的类是否可以使用它 | 从包含它的类派生而来的类是否可以使用它 | 在外部是否可以使用 |
| -------- | ------------------------ | -------------------------------------- | ------------------ |
| 私有部分 | 是                       | 否                                     | 否                 |
| 保护部分 | 是                       | 是                                     | 否                 |
| 公有部分 | 是                       | 是                                     | 是，通过`::`使用   |
