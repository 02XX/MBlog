## 显示模式

### block(块元素)

1. 主题结构标签`<html>` `<body>`
2. 排版标签`<h1>~<h6>` `<hr>` `<p>` `<pre>` `<div>`
3. 列表标签`<ul>` `<ol>` `<li>` `<dl>` `<dt>` `<dd>`
4. 表格相关标签`<table>` `<tbody>` `<thead>` `<tfoot>` `<tr>` `<caption>`
5. `<from>` `<option>`

### inline(行内元素)

1. 文本标签 `<br>` `<em>`
`<strong>` `<sup>` `<sub>` `<del>` `<ins>` 
2. `<a>` `<label>`

### inline-block(行内块元素)

1. 图片`<img>`
2. 单元格 `<td>` `<th>`
3. 表单控件 `<input>` `<textarea>` `<select>` `<button>`
4. 框架标签`<iframe>`
