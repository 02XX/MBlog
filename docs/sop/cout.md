# cout

`cout.precision()`其实是输出流cout的一个格式控制函数，也就是在`iostream`中的一个成员函数。`precision()`返回当前的浮点数的精度值，而`cout.precision(val)`其实就是在输出的时候设定输出值以新的浮点数精度值显示，即小数点后保留val位。
