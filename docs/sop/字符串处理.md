# 字符串处理

```bas
sort [option] filename
	-f	忽略大小写
	-n	以数值型排序
	-r	反向排序
	-t	指定分隔符
	-k n[,m]	按照指定字段范围排序
```

```bash
wc [option] 文件名
	-l：只统计行数
	-w：只统计单词数
	-m：只统计字符串
```
